# NeteaseCloudMusic

## 已实现的功能

- `上拉加载更多`
- `核心功能音乐播放，包括曲目切换、播放、暂停`
- `播放模式切换，单曲循环、列表循环、随机播放`
- `播放列表管理，删除播放列表的歌曲、将歌曲添加到下一首播放`
- `歌曲搜索，实现分页加载，列表必须展示专辑封面图`
- `搜索热词`、`搜索历史的自定义流式布局设计`
- `通知栏音乐控制器，可以控制曲目切换、播放、暂停`
- `封面旋转、可拖动进度条`
